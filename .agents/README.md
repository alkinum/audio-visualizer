# Audio Visualizer Agent Index

This directory is the durable source of truth for the modernization project.

- `STATUS.md`: current implementation state, risks, and evidence
- `ROADMAP.md`: ordered delivery plan and acceptance gates
- `ARCHITECTURE.md`: runtime modules, data flow, and DSP decisions
- `DESIGN.md`: product UI direction, tokens, responsive behavior, and states
- `VERIFICATION.md`: commands and browser checks required before release

## Commit Policy

All commits created during the modernization use exactly three spaces after the
colon:

```text
type:   concise description
```

Allowed types are `docs`, `chore`, `feat`, `fix`, `refactor`, and `test`.
